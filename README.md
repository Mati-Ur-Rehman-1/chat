# Chat App Project

A simple web-based chat application similar to ChatGPT. Users can send messages and receive AI-generated responses.
The project uses a frontend built with HTML/CSS/JS and a backend connected to an AI model deployed on Azure.

---

## **Features**

- Real-time chat interface
- Connects to an AI model hosted on Azure
- Fully deployed on Azure Web App Service
- Simple and responsive frontend
---

## **Tech Stack**

- **Frontend:** HTML, CSS, JavaScript  
- **AI Model:** Azure OpenAI (ChatGPT model deployed via Azure)  
- **Hosting/Deployment:** Azure Web App Service
